Main Roller Fillament

Automatic Roller Filament is a microcontroller-based system using Arduino and a load cell to measure filament weight with high accuracy. Designed for 3D printing workflows, it automates filament preparation by calculating the required grams for each print, reducing manual errors and material waste.

Drone From Stratch

Scratch-Built Arduino Drone is a custom-built UAV developed from the ground up using Arduino as the main flight controller. Integrated with a gyroscope module for attitude stabilization and a GPS module for basic navigation, the drone demonstrates a full-stack approach to hardware assembly, sensor integration, and flight control programming.

Arduino Bluetooth Car

This project involves an Arduino-based Bluetooth-controlled car, which can be operated wirelessly via a Bluetooth application. The car is equipped with an Arduino board and Bluetooth module, allowing users to control movement, speed, and direction through a mobile app.

Obstacle Avoidance Car

This project utilizes an Arduino system integrated with ultrasonic sensors to enable obstacle avoidance functionality. The ultrasonic sensor detects obstacles and obstacles in front of the vehicle, allowing the Arduino to process the data and make real-time decisions to navigate around them.

Autonomus Drone Detection

This project, Autonomous Drone Detection, focuses on enabling a drone to autonomously detect and identify logistics objects that it needs to transport. By utilizing the YOLOv5 algorithm, which is optimized for limited and lightweight resources, the system is designed to efficiently detect objects in real-time. The project aims to ensure high accuracy while minimizing computational load, making it suitable for deployment on resource-constrained drone systems. This approach allows the drone to autonomously perform tasks such as package pickup and delivery, enhancing operational efficiency in logistics applications.

Ground Control Station

This project develops a simplified Ground Control Station (GCS) for real-time drone monitoring and control. It displays essential flight data, including drone heading, GPS status, flight mode, and altitude. The GCS also features action buttons for functions like arming, disarm, RTL, and landing , providing an intuitive interface for efficient and safe drone operation.
